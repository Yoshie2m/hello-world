# CLAUDE.md

このファイルは、このリポジトリでコードを扱う際に Claude Code (claude.ai/code) へ向けたガイダンスを提供します。

## プロジェクト概要

名古屋グルメ（飲食店）情報を整理・発信する静的Webサイト。README.md によると、静的ホスティングを前提としており、バックエンドは持たない。

## 現在の状態

このリポジトリには現時点で計画用ドキュメントのみが存在し、ソースコードはまだ書かれていない（`index.html` / `css/` / `js/` / `data/` は未作成）。`package.json`・ビルドツール・リンター・テストフレームワークも存在しない。下記の構成は「これから作る際の想定レイアウト」であり、実在を確認せずに前提としないこと。

## 想定している技術スタック

- フロントエンドのみ: HTML5 / CSS3 / バニラJS（フレームワーク不使用）
- サーバーサイドの機能なし — すべてブラウザ側（クライアントサイド）で完結させる

## 想定しているフォルダ構成

```
index.html       # メインの画面
css/
  └── style.css  # デザイン・スタイル
js/
  └── app.js     # JSONを読み込んで画面を描画するメイン処理
data/
  └── gourmet.json  # 名古屋グルメのデータ
```

`app.js` は `data/gourmet.json` を読み込み、その内容を `index.html` に描画する想定。HTMLにコンテンツを直書きせず、データ（JSON）と描画ロジック（JS）を分離すること。

## Python利用時の注意

リポジトリ直下に仮想環境 `.venv` が用意されている。ローカルサーバー起動やブラウザ動作確認用ツール（Playwright等）の導入・実行など、Pythonを使う作業は必ずこの `.venv` を有効化した上で行い、システム／グローバルのPython環境に対して `pip install` 等を行わないこと。

```
source .venv/bin/activate
```
